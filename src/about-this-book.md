# About This Book


*Notes on Shaping Life* is a short collection of reflections on attention, action, joy, truth, and the slow work of shaping a life.

It's not a manual, more like a notebook distilled into short chapters.

Some chapters are reminders. Some are questions. Some are arguments with myself written down clearly enough to return to later.

Read it slowly, word by word. If a line hits, pause there.

Author: Luca Pagano
